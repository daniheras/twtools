# TW Tools — Total War Companion

Static companion app for **Total War: Warhammer III**. Built with [Astro](https://astro.build), vanilla TypeScript, and CSS.

## Tools

- **Faction Picker** — Random faction draw with alliance/DLC filters, exclusion pool, and draw history
- **Battle Roller** — *(planned)*
- **Tech Tree** — *(planned)*

## Getting Started

```bash
npm install
npm run dev
```

Open [localhost:4321](http://localhost:4321).

## Build

```bash
npm run build    # production build to dist/
npm run preview  # preview production build
```

## Tech Stack

- **Astro** (SSG, static output)
- **Vanilla TypeScript** for interactivity
- **CSS** with BEM naming and custom properties
